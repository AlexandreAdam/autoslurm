# Legacy Logic Maps

This folder contains dependency flow charts for the current AutoSlurm
implementation.

It is intentionally separate from the new refactor roadmap documents. The
purpose here is to capture how the legacy architecture actually works today so
we can replace it with a cleaner structure later without losing the original
shape of the system.
